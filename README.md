                                                                                                                            DataScience_Task:04

Objective

The objective of this task is to analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

 Dataset Used:

Dataset Name: Twitter Sentiment Training Dataset

Source: Prodigy InfoTech GitHub Repository

File: twitter_training.csv

Columns:

Tweet_ID – Unique identifier for each tweet

Entity – Brand or topic mentioned

Sentiment – Sentiment label (Positive, Negative, Neutral, Irrelevant)

Tweet_Text – Actual tweet content

Tools & Technologies:

Python

Google Colab

Pandas & NumPy

Matplotlib & Seaborn

Regular Expressions (Text Cleaning)

 Steps Performed:

Imported required Python libraries

Loaded and explored the dataset

Assigned column names to the dataset

Handled missing values

Cleaned tweet text (removed URLs, mentions, punctuation, numbers)

Analyzed sentiment distribution

Visualized overall sentiment using bar and pie charts

Performed entity-wise sentiment analysis

Extracted frequently used words from tweets

Results & Insights:

Negative sentiment dominates the dataset, indicating dissatisfaction among users

Positive sentiment reflects brand appreciation and favorable opinions

Neutral sentiment mainly contains informational tweets

Entity-wise sentiment analysis helps understand brand reputation and public perception
